🛵 배달앱

자체 온라인 주문 시스템 만드는 법 — 배달앱 수수료 없이 주문받기

배달앱 수수료 부담 없이 자체 온라인 주문을 받는 방법을 알려드립니다. 무료 도구부터 카카오 주문까지 정리했습니다.
2026-06-20 · 읽는 시간 약 6분 · 배달앱

배달앱 수수료 6~10%는 부담스럽습니다. 단골 손님만큼은 자체 주문 시스템으로 수수료 없이 받을 수 있어요.

📌 최종 업데이트: 2026년 6월 — 최신 정보를 반영했습니다.

자체 주문이 필요한 이유

  • 배달앱 수수료 0원 (결제 수수료만 부담)
  • 단골 손님 데이터 직접 관리 가능
  • 플랫폼 의존도 낮추기
⚠️ 자체 주문은 신규 고객 유입엔 약합니다. 배달앱과 병행하는 게 현실적입니다.

카카오 주문하기 — 가장 쉬운 방법

1
카카오톡 채널 개설
비즈니스 채널 만들고 "주문하기" 기능 신청
2
메뉴 등록
사진, 가격, 옵션 입력 → 카카오페이 결제 자동 연동
3
수수료 확인
배달앱보다 훨씬 낮은 결제 수수료만 부과 (약 2~3%)

네이버 주문 — 플레이스 연동

  • 네이버 플레이스에 "주문하기" 버튼 추가 가능
  • 네이버페이로 결제, 수수료 낮음
  • 플레이스 방문자가 바로 주문 가능

간단한 주문 링크 만들기

💡 무료 방법: 카카오톡 오픈채팅 + 계좌이체로 시작하는 가게도 많습니다. 비용 없이 바로 시작 가능합니다.

자체 주문 홍보 방법

  • 포장 봉투에 "다음엔 카톡으로 주문하면 할인" 스티커
  • 배달앱 리뷰 답글에 "다음엔 직접 주문 환영" 안내
  • 단골에게 직접 채널 추가 요청

정리

  • ✅ 카카오 주문하기 — 가장 쉽고 수수료 저렴
  • ✅ 네이버 플레이스 주문 버튼도 활용
  • ✅ 배달앱과 병행 — 완전 대체는 위험
  • ✅ 단골 고객 대상으로 자체 주문 유도
