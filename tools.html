알바 인건비 계산기
2026년 최저시급 10,320원 기준. 주휴수당·4대보험 포함 총 월 인건비 자동 계산월급 계산기
시급·주 근무시간 → 주휴수당 포함 세전 월급 자동 계산 (2026 최저월급 2,156,880원)시급 계산기
월급·월 근무시간 → 통상시급 역산 + 2026 최저임금(10,320원) 충족 여부 확인실수령액 계산기
세전 월급 → 2026 4대보험 공제 + 홈택스 소득세 반영한 세후 실수령액급여 계산기
기본급·직책·연장·식대 등 항목 합산 → 총급여 + 과세/비과세 구분 (2026 비과세 한도)근무시간 계산기
출근·퇴근·휴게 → 실근무시간(자정 넘김) + 야간·연장 시간과 일급 계산초과근무 계산기
연장근로 시간 → 50% 가산 연장수당 (야간 중복 가산·5인 미만 차이 반영)야간수당 계산기
야간(밤10시~새벽6시) 시간 → 50% 가산 야간수당 (연장 중복·5인 미만 차이 반영)휴일근무수당 계산기
휴일 시간 → 8h이내 1.5배·초과 2배 휴일수당 (야간 중복·5인 미만 차이 반영)연봉↔월급 계산기
연봉↔월급 양방향 변환 + 시급 환산 (연봉÷12=월급, 월급×12=연봉, 세전)일용직 급여 계산기
일당·일수 → 총급여 + 일용직 소득세·지방세 원천징수·실수령 (소액부징수 반영)퇴직금 계산기
입사일·퇴사일·평균임금 → 법정 퇴직금 자동 산출주휴수당 계산기
시급·근무시간 → 주휴수당 지급 여부와 주·월 금액 자동 계산4대보험 계산기
월급 → 직원 공제액과 사업주 부담액 동시 계산 (두루누리 감면 포함)직원 연차 계산기
입사일 → 연차 발생 일수와 미사용 연차수당 자동 계산해고예고수당 계산기
월 통상임금 또는 1일 통상임금 → 근로기준법 제26조 30일분 해고예고수당 자동 계산 (3개월 미만 예외 안내)체불임금 지연이자 계산기
밀린 임금·퇴직금 → 근로기준법 제37조 연 20% 지연이자 자동 계산 (재직·퇴직 기산일 반영, 2025.10.23 개정)배달 순이익 계산기
주문 1건당 수수료·배달비·재료비 빼면 실제 남는 돈배달앱 광고비 ROI 계산기
광고비·추가 주문 → 배달앱 광고 ROI와 손익분기 주문 수 계산메뉴 원가율 계산기
재료비·판매가 → 원가율·마진율·권장 판매가 계산손익분기점(BEP) 계산기
고정비·변동비율 → 최소 목표 매출과 일 매출 목표 계산매장 수익성 계산기
월 매출·비용 → 영업이익률·순이익·목표 손님 수 계산네이버 플레이스 광고비 계산기
예산·CPC·전환율 → 예상 방문 고객과 고객 1명당 광고비 계산부가세 계산기
매출·매입 입력 → 납부세액 또는 환급액 즉시 계산임대료 인상 상한 계산기
현재 월세 → 법정 5% 상한 기준 최대 인상 가능액정책자금 이자 계산기
대출금·금리·기간 → 월 상환액·총이자·총 상환액 계산카드수수료 계산기
월 카드매출 → 우대수수료율 구간 자동 적용, 월·연 수수료 계산권리금 계산기
월 순이익·시설 잔존가치 → 시설+영업 권리금 적정 범위 계산폐업 비용 계산기
철거비·퇴직금·재고 + 폐업 지원금 차감한 실제 부담액 계산창업 초기비용 계산기
업종·평수 → 인테리어·보증금·권리금 등 창업비용 + 월 BEP 매출 추정주식 평단가 계산기
분할매수 매수가·수량으로 평균단가·총매입·평가손익·수익률 자동 계산물타기 계산기
보유 수량·평단가 + 추가매수 → 물타기 후 새 평단가·총수량·평가손익 자동 계산불타기 계산기
보유 수량·평단가 + 상승 추가매수 → 불타기 후 새 평단가·총수량·평가손익 자동 계산주식 수익률 계산기
매수가·매도가·수량 + 수수료·거래세 → 수익금·수익률과 실수익률 자동 계산주식 목표가 계산기
평단가 + 목표 수익률 → 목표 매도가, 손절 수익률 넣으면 손절가·손익비까지 계산미국주식 계산기
달러 매수가·수량 + 실시간 USD/KRW 환율 → 원화 환산 금액·수익률, 환율 효과까지 계산
